# 🚀 Appointment Booking API

A full-featured backend API built with .NET 8 for managing appointments with authentication.

---

## 🔥 Features

- JWT Authentication 🔐
- Create / Update / Delete Appointments
- Protected APIs
- SQL Server (Docker)
- Clean Architecture (Controller → Service → Repository)
- Scalar API UI

---

## 🛠 Tech Stack

- .NET 8 Web API
- Entity Framework Core
- SQL Server (Docker)
- JWT Authentication
- Scalar UI
